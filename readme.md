使用方法
---
	点击`设置`进入设置界面
	`账号`一栏中填入宽带的账号
	`Cookie`一栏中填入的数据需要手动获得:
	打开`HttpWatch`，在路由器的登陆界面输入登录密码，登入的时候，记下HttpWatch记录的`Cookie`,这个`Cookie`就是我们需要的了。

使用注意:
---
	使用时请打开Wifi
	必须安装在用来获取密码的手机上
